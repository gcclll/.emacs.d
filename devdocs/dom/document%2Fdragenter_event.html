Document: dragenter event

The dragenter event is fired when a dragged element or text selection enters a valid drop target.

The target object is the immediate user selection (the element directly indicated by the user as the drop target), or the <body> element.

Bubbles Yes
Cancelable Yes
Default action Reject immediate user selection as potential target element.
Interface DragEvent
Event handler property ondragenter
