URL.username

The username property of the URL interface is a USVString containing the username specified before the domain name.

Note: This feature is available in Web Workers

Value

A USVString.

Examples

const url = new URL('https://anonymous:flabada@developer.mozilla.org/en-US/docs/Web/API/URL/username');
console.log(url.username) // Logs "anonymous"
