DeprecationReportBody.toJSON()

The toJSON() method of the DeprecationReportBody interface is a serializer, and returns a JSON representation of the InterventionReportBody object.

Syntax

toJSON()

Parameters

None.

Return value

A JSON object that is the serialization of the DeprecationReportBody object.

Examples

In this example we create a new ReportingObserver to observe deprecation reports, then return a JSON representation of the first entry.

let options = {
  types: ['deprecation'],
  buffered: true
}

let observer = new ReportingObserver(function(reports, observer) {
  let firstReport = reports[0];
  console.log(firstReport.toJSON());
}, options);
