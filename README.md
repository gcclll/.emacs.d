# keybinds

## F1-F12

| key    | function                                  |
| ------ | ----------------------------------------- |
| `<f6>` | `toggles-hydra/body`，打开 hydra 快捷界面 |

## C-s

| key     | function            |
| ------- | ------------------- |
| `C-s-=` | text scale increase |
| `C-s--` | text scale decrease |
| `C-s-0` | text scale reset    |
