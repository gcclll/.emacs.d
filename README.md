# bindings
## global bindings
| name      | Function              |
| --------- | --------------------- |
| `C-x C-l` | toggle-truncate-lines |
| `C-+`     | text-scale-increase   |
| `C--`     | text-scale-decrease   |
| `M-n`     | forward-paragraph     |
| `M-p`     | backward-paragraph    |
| `M-W =`   | Bigger width          |
| `M-W M-+` | Bigger width          |
| `M-W -`   | smaller width         |
| `M-W M-_` | smaller width         |
| `M-H =`   | Bigger height         |
| `M-H M-+` | bigger height         |
| `M-H -`   | smaller height        |
| `M-H M-_` | smaller height        |
